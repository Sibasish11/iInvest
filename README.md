# iInvest

AI Powered Fintech Dashboard for Smarter Investing

![iInvest Banner](https://via.placeholder.com/1200x300/0a0a1f/ffffff?text=iInvest+-+AI+Powered+Market+Insights)  
*Real-time market pulse, sentiment analysis, event alerts & sector trends*

**iInvest** is a modern, AI driven fintech SaaS platform that helps investors stay ahead of the market with real time insights, sentiment analysis, upcoming economic events, and sector performance tracking all in one beautiful dashboard.

## Live Demo

Experience it live: [https://iinvestmarket.netlify.app](https://iinvestmarket.netlify.app)

## Features

| Feature                     | Description                                                                 |
|-----------------------------|-----------------------------------------------------------------------------|
| **Market Pulse Dashboard**  | Real-time overview with current sentiment, next big events & top sectors    |
| **Sentiment Analysis**      | Instantly see if the market is Optimistic, Neutral, or Cautious             |
| **Event Alerts**            | Countdown to major events like FOMC meetings, earnings, etc.                |
| **Sector Trends**           | Track outperformers (e.g., +4.1% Energy) and underperformers                |
| **AI Assistant**            | Chat with Google's Gemini-powered AI for instant financial insights         |
| **Tabs Navigation**         | Clean interface: Overview, AI Assistant, Insights, Knowledge base           |
| **Real-time / Historical**  | Switch between live data and historical views                               |

## Tech Stack

- **Frontend**: React + TypeScript + Vite  
- **AI Integration**: Google Gemini API  
- **Styling**: Tailwind CSS (or your preferred framework)  
- **Deployment**: Netlify  
- **Date Format**: Works with current date (e.g., January 05, 2026)

## Quick Start (Local Development)

### Prerequisites
- Node.js v18+
- Google Gemini API key (get it free from [Google AI Studio](https://aistudio.google.com/))

### Steps
1. Clone the repo:
   ```bash
   git clone https://github.com/Sibasish11/iInvest.git
   cd iInvest
   ```

### Install dependencies:Bashnpm install

Create .env.local and add your API key:envVITE_GEMINI_API_KEY=your_gemini_api_key_here
Start the development server:Bashnpm run dev
Open http://localhost:5173 in your browser
